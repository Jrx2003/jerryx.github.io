| 类型 | 问题 | 难度 |
|------|------|------|
| **新用户** | 无历史行为 | ⭐⭐⭐ |
| **新物品** | 无交互数据 | ⭐⭐⭐ |
| **新系统** | 无数据积累 | ⭐⭐⭐⭐⭐ |

## 二、新用户冷启动

### 策略

1. **热门推荐**
   - 全站热门
   - 地域热门
   - 时段热门

2. **内容引导**
   - 兴趣标签选择
   - 偏好问卷
   - 社交登录导入

3. **迁移学习**
   - 相似用户推荐
   - 人口统计学推荐
   - 设备/位置推荐

### 实现

```python
def cold_start_for_new_user(user_info):
    """新用户推荐策略"""

    # 1. 基于人口统计学
    if user_info.get('age') and user_info.get('gender'):
        demo_recs = get_popular_by_demographics(
            age=user_info['age'],
            gender=user_info['gender'],
            n=10
        )

    # 2. 基于位置
    if user_info.get('location'):
        location_recs = get_trending_in_location(
            user_info['location'],
            n=10
        )

    # 3. 全局热门
    trending_recs = get_global_trending(n=10)

    # 4. 混合
    final_recs = merge_recommendations(
        [demo_recs, location_recs, trending_recs],
        weights=[0.3, 0.3, 0.4]
    )

    return final_recs
```

## 三、新物品冷启动

### 策略

1. **内容-based**
   - 基于物品属性
   - 基于文本/图像
   - 基于类别

2. **探索利用 (EE)**
   - 多臂老虎机
   - Thompson Sampling
   - UCB

3. **社交关系**
   - 作者粉丝推送
   - 相似物品推荐

### 实现

```python
def cold_start_for_new_item(item_info):
    """新物品冷启动"""

    # 1. 内容相似推荐
    similar_items = find_similar_by_content(item_info)

    # 2. 新物品探索池
    exploration_boost = 1.5  # 提高曝光概率

    # 3. 作者已有粉丝
    if item_info.get('author_id'):
        author_fans = get_author_fans(item_info['author_id'])
        push_to_users(author_fans, item_info)

    return exploration_boost
```
