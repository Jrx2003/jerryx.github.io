在线学习概述

什么是在线学习

在线学习 (Online Learning) 指模型能够根据实时反馈持续更新。

传统离线学习:
数据收集 → 批量训练 → 模型发布 → 周期更新 (天/周)

在线学习:
实时数据 → 增量更新 → 实时生效 (秒/分钟)

应用场景

场景说明
新闻推荐热点快速变化
直播推荐实时性要求高
广告投放市场变化快
金融风控欺诈模式变化

在线算法

FTRL

class FTRL:
    """FTRL 优化器"""

    def __init__(self, alpha=0.005, beta=1.0, l1=0.1, l2=1.0):
        self.alpha = alpha
        self.beta = beta
        self.l1 = l1
        self.l2 = l2

        self.z = {}  # 累积梯度
        self.n = {}  # 梯度平方累积

    def update(self, weights, gradients):
        """参数更新"""
        for feature_id, grad in gradients.items():
            if feature_id not in self.z:
                self.z[feature_id] = 0.0
                self.n[feature_id] = 0.0

            self.n[feature_id] += grad ** 2

            sigma = (math.sqrt(self.n[feature_id]) -
                     math.sqrt(self.n[feature_id] - grad ** 2)) / self.alpha

            self.z[feature_id] += grad - sigma * weights.get(feature_id, 0)

            # 计算新权重
            if abs(self.z[feature_id]) <= self.l1:
                weights[feature_id] = 0.0
            else:
                weights[feature_id] = (
                    (self.l1 * (1 if self.z[feature_id] >= 0 else -1) -
                     self.z[feature_id]) /
                    ((self.beta + math.sqrt(self.n[feature_id])) / self.alpha +
                     self.l2)
                )

        return weights

Bandit 算法

class LinUCB:
    """LinUCB 上下文老虎机"""

    def __init__(self, n_features, alpha=1.0):
        self.alpha = alpha
        self.n_features = n_features

        self.A = {}  # arm_id -> (d, d) matrix
        self.b = {}  # arm_id -> (d,) vector

    def select_arm(self, context, arm_ids):
        """选择 arm"""
        best_arm = None
        best_score = -float('inf')

        for arm_id in arm_ids:
            if arm_id not in self.A:
                self.A[arm_id] = np.eye(self.n_features)
                self.b[arm_id] = np.zeros(self.n_features)

            theta = np.linalg.solve(self.A[arm_id], self.b[arm_id])

            p = np.dot(theta, context)
            cb = self.alpha * np.sqrt(
                np.dot(context, np.linalg.solve(self.A[arm_id], context))
            )
            score = p + cb

            if score > best_score:
                best_score = score
                best_arm = arm_id

        return best_arm

    def update(self, arm_id, context, reward):
        """更新 arm 的参数"""
        self.A[arm_id] += np.outer(context, context)
        self.b[arm_id] += reward * context

面试要点

高频问题

Q1: 在线学习和离线学习的区别？

A:

维度离线学习在线学习
更新频率天/周秒/分钟
数据使用批量流式
适应性慢快

Q2: FTRL 为什么适合在线学习？

A:

  • 适合稀疏特征
  • 产生稀疏解
  • 不需要保存历史数据
  • 每样本更新

Q3: 如何保证在线学习的稳定性？

A:

  1. 梯度裁剪
  2. 学习率衰减
  3. A/B 测试验证
  4. 监控和回滚机制
