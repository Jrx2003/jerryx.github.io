## 跨域推荐概述

### 什么是跨域推荐

利用源领域的知识帮助目标领域推荐。

```
场景:
- 电商 → 视频
- 新闻 → 广告
- 国内 → 海外

挑战:
- 领域差异
- 用户重叠度
- 物品差异
```

### 类型

| 类型 | 说明 |
|------|------|
| 用户重叠 | 两领域有共同用户 |
| 物品重叠 | 两领域有共同物品 |
| 无重叠 | 完全独立的领域 |

---

## 迁移学习方法

### 共享表示

```python
class CrossDomainModel(nn.Module):
    def __init__(self):
        # 共享编码器
        self.shared_encoder = nn.Linear(input_dim, hidden_dim)

        # 领域特定塔
        self.source_tower = DomainTower(hidden_dim)
        self.target_tower = DomainTower(hidden_dim)

    def forward(self, x, domain):
        shared = self.shared_encoder(x)
        if domain == 'source':
            return self.source_tower(shared)
        else:
            return self.target_tower(shared)
```

## 元学习应用

### MAML

```python
class MAML:
    """Model-Agnostic Meta-Learning"""

    def meta_train(self, tasks):
        for task in tasks:
            # 内循环: 任务适应
            adapted_params = self.inner_loop(task)

            # 外循环: 元更新
            meta_loss = self.compute_loss(adapted_params, task)
            meta_loss.backward()
```
