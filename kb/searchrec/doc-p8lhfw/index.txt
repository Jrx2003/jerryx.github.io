## 语义搜索概述

### 什么是语义搜索

语义搜索理解查询的**含义**，而非仅仅匹配**关键词**。

```
关键词搜索:
查询: "苹果 价格"
匹配: 包含 "苹果" 和 "价格" 的文档
问题: 无法区分 "苹果手机" 和 "水果苹果"

语义搜索:
查询: "苹果 价格"
理解: 根据上下文理解用户意图
匹配: 语义相近的文档
```

### 语义搜索架构

```
用户查询
    │
    ▼
┌─────────────────────────────────────────────────────────────┐
│                     查询理解层                               │
└─────────────────────────────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────────────────────────────┐
│                     向量表示层                               │
│  - 查询编码: Query → Embedding                              │
│  - 文档编码: Doc → Embedding (离线)                          │
└─────────────────────────────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────────────────────────────┐
│                     向量检索层                               │
│  - 相似度计算 (余弦相似度、点积)                             │
│  - ANN 近似检索 (FAISS、HNSW、Milvus)                        │
└─────────────────────────────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────────────────────────────┐
│                     结果排序层                               │
└─────────────────────────────────────────────────────────────┘
```

## 文本表示方法

### 深度学习方法

```python
import torch
import torch.nn as nn
from transformers import AutoTokenizer, AutoModel

class TextEncoder:
    """文本编码器"""

    def __init__(self, model_name='bert-base-chinese'):
        self.tokenizer = AutoTokenizer.from_pretrained(model_name)
        self.model = AutoModel.from_pretrained(model_name)
        self.model.eval()

    @torch.no_grad()
    def encode(self, texts, batch_size=32, normalize=True):
        embeddings = []

        for i in range(0, len(texts), batch_size):
            batch_texts = texts[i:i + batch_size]

            inputs = self.tokenizer(
                batch_texts,
                padding=True,
                truncation=True,
                max_length=512,
                return_tensors='pt'
            )

            outputs = self.model(**inputs)
            batch_embeddings = outputs.last_hidden_state[:, 0]

            if normalize:
                batch_embeddings = nn.functional.normalize(
                    batch_embeddings, p=2, dim=1
                )

            embeddings.append(batch_embeddings.cpu())

        return torch.cat(embeddings, dim=0)
```

## 向量检索基础

### 向量相似度

```python
import torch
import torch.nn.functional as F

def cosine_similarity(a, b):
    """余弦相似度"""
    return F.cosine_similarity(a.unsqueeze(0), b.unsqueeze(0))

def dot_product_similarity(a, b):
    """点积相似度"""
    return torch.dot(a, b)
```

## 近似最近邻搜索

### HNSW

```python
import hnswlib

class HNSWIndex:
    """HNSW 索引"""

    def __init__(self, dimension, max_elements=1000000):
        self.index = hnswlib.Index(space='cosine', dim=dimension)
        self.index.init_index(
            max_elements=max_elements,
            ef_construction=200,
            M=16
        )

    def add(self, vectors, ids=None):
        if ids is None:
            ids = np.arange(len(vectors))
        self.index.add_items(vectors, ids)

    def search(self, query, top_k=10, ef=50):
        self.index.set_ef(ef)
        labels, distances = self.index.knn_query(query, k=top_k)
        return labels[0], distances[0]
```

A:
| 维度 | 关键词搜索 | 语义搜索 |
|------|------------|----------|
| 匹配方式 | 字面匹配 | 语义匹配 |
| 理解能力 | 无 | 理解同义词、上下文 |
| 召回率 | 低 | 高 |
| 计算成本 | 低 | 高 |

A:
| 类型 | 代表算法 | 特点 |
|------|----------|------|
| 基于树 | Annoy | 简单，召回率一般 |
| 基于哈希 | LSH | 内存小，精度低 |
| 基于量化 | PQ, OPQ | 内存高效 |
| 基于图 | HNSW | 查询快，内存大 |
| 基于聚类 | IVF | 平衡速度和精度 |
