在线学习概述

什么是在线学习

模型根据实时反馈持续更新。

离线学习: 批量训练 → 定期更新 (天/周)
在线学习: 实时数据 → 增量更新 (秒/分钟)

应用场景

  • 新闻推荐 (热点变化快)
  • 实时广告
  • 金融风控

FTRL 算法

Follow The Regularized Leader

class FTRL:
    def __init__(self, alpha=0.005, beta=1.0, l1=0.1, l2=1.0):
        self.alpha = alpha
        self.beta = beta
        self.l1 = l1
        self.l2 = l2
        self.z = {}
        self.n = {}

    def update(self, weights, gradients):
        for f, g in gradients.items():
            self.n[f] = self.n.get(f, 0) + g ** 2
            sigma = (math.sqrt(self.n[f]) - math.sqrt(self.n[f] - g**2)) / self.alpha
            self.z[f] = self.z.get(f, 0) + g - sigma * weights.get(f, 0)

            if abs(self.z[f]) <= self.l1:
                weights[f] = 0
            else:
                weights[f] = (self.l1 * (1 if self.z[f] > 0 else -1) - self.z[f]) / \
                             ((self.beta + math.sqrt(self.n[f])) / self.alpha + self.l2)
        return weights

实时更新策略

参数服务器

Worker: 计算梯度 → Push 到 Server
Server: 聚合更新 → 同步给 Workers

面试要点

Q1: FTRL 的优点？

A:

  • 适合稀疏特征
  • 产生稀疏解
  • 每样本更新

Q2: 在线学习的挑战？

A:

  • 稳定性
  • 冷启动
  • 评估困难
