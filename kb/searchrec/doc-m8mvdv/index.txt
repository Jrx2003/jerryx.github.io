## 多样性概述

### 为什么需要多样性

```
问题:
- 结果同质化严重
- 信息茧房
- 用户兴趣狭窄

目标:
- 覆盖多个兴趣点
- 发现新领域
- 提升用户留存
```

### 多样性类型

| 类型 | 说明 |
|------|------|
| 个体多样性 | 单个用户结果的多样性 |
| 群体多样性 | 所有用户结果的多样性 |

## MMR 算法

### 最大边际相关性

```python
def mmr(query, candidates, lambda_param=0.5, top_k=10):
    """
    MMR = λ * Relevance - (1-λ) * MaxSimilarity
    """
    selected = []

    while len(selected) < top_k and candidates:
        if not selected:
            # 第一个选最相关的
            best = max(candidates, key=lambda x: x.relevance)
        else:
            def score(item):
                rel = item.relevance
                max_sim = max(similarity(item, s) for s in selected)
                return lambda_param * rel - (1 - lambda_param) * max_sim

            best = max(candidates, key=score)

        selected.append(best)
        candidates.remove(best)

    return selected
```

## DPP 行列式点过程

### 原理

```
DPP: Determinantal Point Process

特点:
- 天然多样性建模
- 考虑物品间相似度
- 数学优雅
```

### 实现

```python
def dpp_sample(kernel_matrix, max_length):
    """
    kernel_matrix: 物品间的核矩阵
    """
    selected = []
    candidates = list(range(len(kernel_matrix)))

    while len(selected) < max_length and candidates:
        # 计算边际增益
        scores = []
        for c in candidates:
            # 计算条件概率
            prob = compute_conditional_prob(kernel_matrix, selected, c)
            scores.append(prob)

        # 选择得分最高的
        best_idx = candidates[np.argmax(scores)]
        selected.append(best_idx)
        candidates.remove(best_idx)

    return selected
```

## 新颖性优化

### 策略

```python
def novelty_score(item, user_history, global_popularity):
    """
    新颖性 = 1 / (用户熟悉度 + 全局流行度)
    """
    user_familiarity = 1 if item in user_history else 0
    return 1 / (1 + user_familiarity + global_popularity[item])
```
