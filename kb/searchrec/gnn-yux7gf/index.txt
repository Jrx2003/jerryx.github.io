## GNN 概述

### 什么是 GNN

图神经网络 (Graph Neural Network) 处理图结构数据。

```
图结构: G = (V, E)
- V: 节点 (用户、物品)
- E: 边 (交互关系)

应用场景:
- 社交推荐
- 知识图谱
- 分子结构
```

### 消息传递

```
核心思想:
1. 聚合邻居信息
2. 更新节点表示
3. 重复多轮
```

---

## 图卷积网络 (GCN)

### 原理

```
H^(l+1) = σ(D^(-1/2) A D^(-1/2) H^(l) W^(l))

- A: 邻接矩阵
- D: 度矩阵
- H: 节点特征
```

### 实现

```python
class GCNLayer(nn.Module):
    def __init__(self, in_features, out_features):
        super().__init__()
        self.linear = nn.Linear(in_features, out_features)

    def forward(self, x, adj):
        x = torch.matmul(adj, x)
        x = self.linear(x)
        return x
```

## GraphSAGE

### 特点

- 采样邻居而非全图
- 支持归纳学习
- 可扩展到大图

### 实现

```python
class GraphSAGELayer(nn.Module):
    def __init__(self, in_features, out_features):
        super().__init__()
        self.linear = nn.Linear(in_features * 2, out_features)

    def forward(self, x, neighbors):
        # 聚合邻居
        neighbor_agg = torch.mean(neighbors, dim=1)
        # 拼接自身和邻居
        combined = torch.cat([x, neighbor_agg], dim=1)
        return self.linear(combined)
```

---

## 图注意力网络 (GAT)

### 特点

- 为不同邻居分配不同权重
- 使用注意力机制

### 实现

```python
class GATLayer(nn.Module):
    def __init__(self, in_features, out_features, num_heads=8):
        super().__init__()
        self.attention = nn.MultiheadAttention(out_features, num_heads)

    def forward(self, x, adj):
        # 计算注意力
        attn_output, _ = self.attention(x, x, x, attn_mask=adj)
        return attn_output
```

## 面试要点

**Q1: GNN 和 CNN 的区别？**

A:
- CNN 处理规则网格数据
- GNN 处理不规则图数据
- GNN 需要处理邻居聚合

**Q2: GCN 和 GraphSAGE 的区别？**

A:
- GCN 使用全图
- GraphSAGE 采样邻居
- GraphSAGE 支持归纳学习
