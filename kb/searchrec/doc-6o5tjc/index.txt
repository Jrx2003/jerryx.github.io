## 矩阵分解概述

矩阵分解（Matrix Factorization, MF）是协同过滤的一种基于模型的方法。其核心思想是：**将高维的用户-物品评分矩阵分解为低维的隐因子矩阵**。

### 基本思想

```
原始评分矩阵 R (m×n)          低维分解
┌─────────────────┐           ┌──────────┐   ┌──────────┐
│ 5   3   ?   1   │           │ 0.8  0.2 │   │ 0.9  0.1 │
│ 4   ?   4   2   │    ≈      │ 0.6  0.4 │ × │ 0.3  0.7 │
│ ?   5   3   ?   │           │ 0.2  0.8 │   │ 0.5  0.5 │
│ 2   1   ?   5   │           │ 0.9  0.1 │   │ 0.2  0.8 │
└─────────────────┘           └──────────┘   └──────────┘
                                    P (m×k)      Q^T (k×n)
```

其中：
- $m$：用户数量
- $n$：物品数量
- $k$：隐因子维度（通常 $k \ll \min(m,n)$）

### 预测公式

$$\hat{r}_{ui} = p_u^T q_i = \sum_{f=1}^{k} p_{uf} \cdot q_{if}$$

其中：
- $p_u \in \mathbb{R}^k$：用户$u$的隐向量
- $q_i \in \mathbb{R}^k$：物品$i$的隐向量

---

## 奇异值分解 (SVD)

### 数学定义

对于矩阵 $R \in \mathbb{R}^{m \times n}$，SVD 分解为：

$$R = U \Sigma V^T$$

其中：
- $U \in \mathbb{R}^{m \times m}$：左奇异向量（用户特征）
- $\Sigma \in \mathbb{R}^{m \times n}$：对角矩阵，对角线为奇异值
- $V \in \mathbb{R}^{n \times n}$：右奇异向量（物品特征）

### 截断 SVD

取前 $k$ 个最大的奇异值：

$$R \approx U_k \Sigma_k V_k^T = P \cdot Q^T$$

### 带偏置的 SVD

$$\hat{r}_{ui} = \mu + b_u + b_i + p_u^T q_i$$

其中：
- $\mu$：全局平均评分
- $b_u$：用户偏置（用户评分习惯）
- $b_i$：物品偏置（物品整体质量）

---

## 交替最小二乘法 (ALS)

### 算法思想

ALS 通过交替固定一个矩阵来优化另一个矩阵，将非凸优化转化为凸优化问题。

### 损失函数

$$\min_{P,Q} \sum_{(u,i) \in K} (r_{ui} - p_u^T q_i)^2 + \lambda (\|p_u\|^2 + \|q_i\|^2)$$

其中：
- $K$：观测到的评分集合
- $\lambda$：正则化系数

### 求解过程

**固定 $Q$，优化 $P$：**

对每个用户 $u$：
$$p_u = (Q^T Q + \lambda I)^{-1} Q^T r_u$$

**固定 $P$，优化 $Q$：**

对每个物品 $i$：
$$q_i = (P^T P + \lambda I)^{-1} P^T r_i$$

### 算法流程

```
1. 随机初始化 P 和 Q
2. 重复直到收敛：
   a. 固定 Q，更新所有 p_u
   b. 固定 P，更新所有 q_i
3. 返回 P 和 Q
```

### 代码实现

```python
import numpy as np

class ALS:
    def __init__(self, n_factors=20, n_epochs=20, reg=0.1):
        self.n_factors = n_factors
        self.n_epochs = n_epochs
        self.reg = reg

    def fit(self, ratings):
        """
        ratings: 用户-物品评分矩阵 (m×n)，缺失值为0
        """
        m, n = ratings.shape

        # 初始化隐向量
        self.P = np.random.normal(0, 0.1, (m, self.n_factors))
        self.Q = np.random.normal(0, 0.1, (n, self.n_factors))

        # 创建掩码矩阵（标记已知评分）
        mask = (ratings > 0).astype(float)

        for epoch in range(self.n_epochs):
            # 固定 Q，更新 P
            for u in range(m):
                items = np.where(mask[u] > 0)[0]
                if len(items) == 0:
                    continue
                Q_subset = self.Q[items]
                r_subset = ratings[u, items]

                A = Q_subset.T @ Q_subset + self.reg * np.eye(self.n_factors)
                b = Q_subset.T @ r_subset
                self.P[u] = np.linalg.solve(A, b)

            # 固定 P，更新 Q
            for i in range(n):
                users = np.where(mask[:, i] > 0)[0]
                if len(users) == 0:
                    continue
                P_subset = self.P[users]
                r_subset = ratings[users, i]

                A = P_subset.T @ P_subset + self.reg * np.eye(self.n_factors)
                b = P_subset.T @ r_subset
                self.Q[i] = np.linalg.solve(A, b)

    def predict(self, u, i):
        return self.P[u] @ self.Q[i]
```

## 随机梯度下降 (SGD)

### 损失函数

$$L = \sum_{(u,i) \in K} (r_{ui} - \hat{r}_{ui})^2 + \lambda (\|p_u\|^2 + \|q_i\|^2)$$

### 梯度更新

$$e_{ui} = r_{ui} - \hat{r}_{ui}$$

$$p_u \leftarrow p_u + \gamma (e_{ui} \cdot q_i - \lambda p_u)$$

$$q_i \leftarrow q_i + \gamma (e_{ui} \cdot p_u - \lambda q_i)$$

其中 $\gamma$ 为学习率。

### 带偏置的 SGD

$$\hat{r}_{ui} = \mu + b_u + b_i + p_u^T q_i$$

梯度更新：
- $b_u \leftarrow b_u + \gamma (e_{ui} - \lambda b_u)$
- $b_i \leftarrow b_i + \gamma (e_{ui} - \lambda b_i)$

## 因子分解机 (FM)

### 问题背景

传统线性模型：
$$\hat{y} = w_0 + \sum_{i=1}^n w_i x_i$$

无法捕捉特征交叉。多项式模型：
$$\hat{y} = w_0 + \sum_{i=1}^n w_i x_i + \sum_{i=1}^n \sum_{j=i+1}^n w_{ij} x_i x_j$$

参数数量 $O(n^2)$，稀疏数据下难以学习。

### FM 核心思想

用隐向量内积表示交叉项权重：
$$w_{ij} := v_i^T v_j$$

### FM 公式

$$\hat{y} = w_0 + \sum_{i=1}^n w_i x_i + \sum_{i=1}^n \sum_{j=i+1}^n v_i^T v_j \cdot x_i x_j$$

其中：
- $w_0$：全局偏置
- $w_i$：一阶特征权重
- $v_i \in \mathbb{R}^k$：第$i$个特征的隐向量

### 复杂度优化

直接计算二阶项复杂度为 $O(kn^2)$，通过数学变换可优化到 $O(kn)$：

$$\sum_{i=1}^n \sum_{j=i+1}^n v_i^T v_j \cdot x_i x_j = \frac{1}{2} \sum_{f=1}^k \left[ \left( \sum_{i=1}^n v_{if} x_i \right)^2 - \sum_{i=1}^n v_{if}^2 x_i^2 \right]$$

### 时间复杂度对比

| 模型 | 训练复杂度 | 预测复杂度 |
|------|------------|------------|
| 多项式 | $O(n^2)$ | $O(n^2)$ |
| FM | $O(kn)$ | $O(kn)$ |

### FM 特点

| 优点 | 说明 |
|------|------|
| 自动特征交叉 | 自动学习所有二阶特征组合 |
| 参数高效 | 参数数量 $O(kn)$，可处理高维稀疏数据 |
| 泛化能力强 | 未出现的特征组合可通过隐向量内积计算 |
| 适用性广 | 可用于回归、分类、排序等任务 |

## 场感知因子分解机 (FFM)

### 核心思想

每个特征对每个场（field）学习不同的隐向量。

### FFM 公式

$$\hat{y} = w_0 + \sum_{i=1}^n w_i x_i + \sum_{i=1}^n \sum_{j=i+1}^n v_{i,f_j}^T v_{j,f_i} \cdot x_i x_j$$

其中：
- $f_i$：特征$i$所属的场
- $v_{i,f_j}$：特征$i$对场$f_j$的隐向量

### FM vs FFM

| 维度 | FM | FFM |
|------|-----|-----|
| 隐向量 | 每个特征一个隐向量 | 每个特征对每个场一个隐向量 |
| 参数数量 | $O(kn)$ | $O(knF)$，$F$为场数 |
| 表达能力 | 较弱 | 更强，考虑场信息 |
| 训练速度 | 快 | 慢（约慢10倍） |
| 适用场景 | 特征数多、场数少 | 场数多、特征间关系复杂 |

### 场的定义

```
示例：广告点击预测

特征：
- Publisher: ESPN, Vogue, NBC
- Advertiser: Nike, Gucci, Adidas
- Gender: Male, Female

场划分：
- Publisher 场：ESPN, Vogue, NBC
- Advertiser 场：Nike, Gucci, Adidas
- Gender 场：Male, Female

FFM 隐向量：
- ESPN 对 Advertiser 场的隐向量
- ESPN 对 Gender 场的隐向量
- Nike 对 Publisher 场的隐向量
- ...
```

## 深度因子分解机 (DeepFM)

### 架构

DeepFM 结合了 FM 的浅层特征交叉能力和 DNN 的深层特征学习能力：

```
输入特征
    ↓
┌─────────────────────────────────────┐
│              FM 部分                 │
│  一阶项 + 二阶交叉项（内积）          │
└─────────────────────────────────────┘
          ↓
┌─────────────────────────────────────┐
│              DNN 部分                │
│  Embedding → 多层全连接网络          │
└─────────────────────────────────────┘
          ↓
      FM输出 + DNN输出 → Sigmoid
```

### 公式

$$\hat{y} = \sigma(y_{FM} + y_{DNN})$$

其中：
$$y_{FM} = w_0 + \sum_{i=1}^n w_i x_i + \sum_{i=1}^n \sum_{j=i+1}^n v_i^T v_j \cdot x_i x_j$$

$$y_{DNN} = W^{|L|+1} \cdot a^L + b^{|L|+1}$$

### 特点

- **端到端训练**：FM 和 DNN 共享 Embedding
- **低阶+高阶**：同时学习低阶和高阶特征交叉
- **无需特征工程**：自动学习特征交叉

## 模型对比总结

| 模型 | 核心思想 | 特征交叉 | 适用场景 | 复杂度 |
|------|----------|----------|----------|--------|
| **SVD** | 矩阵分解 | 无 | 评分预测 | 低 |
| **ALS** | 交替优化 | 无 | 大规模评分预测 | 中 |
| **FM** | 隐向量内积 | 二阶自动交叉 | CTR预估、排序 | 低 |
| **FFM** | 场感知隐向量 | 二阶场感知交叉 | 场信息重要时 | 高 |
| **DeepFM** | FM + DNN | 低阶+高阶交叉 | 复杂特征交互 | 高 |

## 工业实践要点

### 1. 隐向量维度选择

| 数据规模 | 推荐维度 | 说明 |
|----------|----------|------|
| 小数据集 | 10-20 | 避免过拟合 |
| 中等数据集 | 50-100 | 平衡表达能力和泛化 |
| 大数据集 | 100-500 | 充分学习复杂模式 |

### 2. 正则化策略

- **L2正则化**：最常用，控制参数大小
- **Dropout**：DNN部分使用
- **Early Stopping**：防止过拟合

### 3. 负采样

对于隐式反馈数据（如点击），需要负采样：

```python
# 负采样示例
for user in users:
    # 正样本：用户交互过的物品
    pos_items = user_interactions[user]

    # 负样本：随机采样未交互的物品
    neg_items = random.sample(all_items - pos_items, k=len(pos_items))
```

### 基本思想

```
原始评分矩阵 R (m×n)          低维分解
┌─────────────────┐           ┌──────────┐   ┌──────────┐
│ 5   3   ?   1   │           │ 0.8  0.2 │   │ 0.9  0.1 │
│ 4   ?   4   2   │    ≈      │ 0.6  0.4 │ × │ 0.3  0.7 │
│ ?   5   3   ?   │           │ 0.2  0.8 │   │ 0.5  0.5 │
│ 2   1   ?   5   │           │ 0.9  0.1 │   │ 0.2  0.8 │
└─────────────────┘           └──────────┘   └──────────┘
                                    P (m×k)      Q^T (k×n)
```

其中：
- $m$：用户数量
- $n$：物品数量
- $k$：隐因子维度（通常 $k \ll \min(m,n)$）

### 预测公式

$$\hat{r}_{ui} = p_u^T q_i = \sum_{f=1}^{k} p_{uf} \cdot q_{if}$$

其中：
- $p_u \in \mathbb{R}^k$：用户$u$的隐向量
- $q_i \in \mathbb{R}^k$：物品$i$的隐向量

---

## 奇异值分解 (SVD)

### 数学定义

对于矩阵 $R \in \mathbb{R}^{m \times n}$，SVD 分解为：

$$R = U \Sigma V^T$$

其中：
- $U \in \mathbb{R}^{m \times m}$：左奇异向量（用户特征）
- $\Sigma \in \mathbb{R}^{m \times n}$：对角矩阵，对角线为奇异值
- $V \in \mathbb{R}^{n \times n}$：右奇异向量（物品特征）

### 截断 SVD

取前 $k$ 个最大的奇异值：

$$R \approx U_k \Sigma_k V_k^T = P \cdot Q^T$$

### 带偏置的 SVD

$$\hat{r}_{ui} = \mu + b_u + b_i + p_u^T q_i$$

其中：
- $\mu$：全局平均评分
- $b_u$：用户偏置（用户评分习惯）
- $b_i$：物品偏置（物品整体质量）

---

## 交替最小二乘法 (ALS)

### 算法思想

ALS 通过交替固定一个矩阵来优化另一个矩阵，将非凸优化转化为凸优化问题。

### 损失函数

$$\min_{P,Q} \sum_{(u,i) \in K} (r_{ui} - p_u^T q_i)^2 + \lambda (\|p_u\|^2 + \|q_i\|^2)$$

其中：
- $K$：观测到的评分集合
- $\lambda$：正则化系数

### 求解过程

**固定 $Q$，优化 $P$：**

对每个用户 $u$：
$$p_u = (Q^T Q + \lambda I)^{-1} Q^T r_u$$

**固定 $P$，优化 $Q$：**

对每个物品 $i$：
$$q_i = (P^T P + \lambda I)^{-1} P^T r_i$$

### 算法流程

```
1. 随机初始化 P 和 Q
2. 重复直到收敛：
   a. 固定 Q，更新所有 p_u
   b. 固定 P，更新所有 q_i
3. 返回 P 和 Q
```

### 代码实现

```python
import numpy as np

class ALS:
    def __init__(self, n_factors=20, n_epochs=20, reg=0.1):
        self.n_factors = n_factors
        self.n_epochs = n_epochs
        self.reg = reg

    def fit(self, ratings):
        """
        ratings: 用户-物品评分矩阵 (m×n)，缺失值为0
        """
        m, n = ratings.shape

        # 初始化隐向量
        self.P = np.random.normal(0, 0.1, (m, self.n_factors))
        self.Q = np.random.normal(0, 0.1, (n, self.n_factors))

        # 创建掩码矩阵（标记已知评分）
        mask = (ratings > 0).astype(float)

        for epoch in range(self.n_epochs):
            # 固定 Q，更新 P
            for u in range(m):
                items = np.where(mask[u] > 0)[0]
                if len(items) == 0:
                    continue
                Q_subset = self.Q[items]
                r_subset = ratings[u, items]

                A = Q_subset.T @ Q_subset + self.reg * np.eye(self.n_factors)
                b = Q_subset.T @ r_subset
                self.P[u] = np.linalg.solve(A, b)

            # 固定 P，更新 Q
            for i in range(n):
                users = np.where(mask[:, i] > 0)[0]
                if len(users) == 0:
                    continue
                P_subset = self.P[users]
                r_subset = ratings[users, i]

                A = P_subset.T @ P_subset + self.reg * np.eye(self.n_factors)
                b = P_subset.T @ r_subset
                self.Q[i] = np.linalg.solve(A, b)

    def predict(self, u, i):
        return self.P[u] @ self.Q[i]
```

## 随机梯度下降 (SGD)

### 损失函数

$$L = \sum_{(u,i) \in K} (r_{ui} - \hat{r}_{ui})^2 + \lambda (\|p_u\|^2 + \|q_i\|^2)$$

### 梯度更新

$$e_{ui} = r_{ui} - \hat{r}_{ui}$$

$$p_u \leftarrow p_u + \gamma (e_{ui} \cdot q_i - \lambda p_u)$$

$$q_i \leftarrow q_i + \gamma (e_{ui} \cdot p_u - \lambda q_i)$$

其中 $\gamma$ 为学习率。

### 带偏置的 SGD

$$\hat{r}_{ui} = \mu + b_u + b_i + p_u^T q_i$$

梯度更新：
- $b_u \leftarrow b_u + \gamma (e_{ui} - \lambda b_u)$
- $b_i \leftarrow b_i + \gamma (e_{ui} - \lambda b_i)$

## 因子分解机 (FM)

### 问题背景

传统线性模型：
$$\hat{y} = w_0 + \sum_{i=1}^n w_i x_i$$

无法捕捉特征交叉。多项式模型：
$$\hat{y} = w_0 + \sum_{i=1}^n w_i x_i + \sum_{i=1}^n \sum_{j=i+1}^n w_{ij} x_i x_j$$

参数数量 $O(n^2)$，稀疏数据下难以学习。

### FM 核心思想

用隐向量内积表示交叉项权重：
$$w_{ij} := v_i^T v_j$$

### FM 公式

$$\hat{y} = w_0 + \sum_{i=1}^n w_i x_i + \sum_{i=1}^n \sum_{j=i+1}^n v_i^T v_j \cdot x_i x_j$$

其中：
- $w_0$：全局偏置
- $w_i$：一阶特征权重
- $v_i \in \mathbb{R}^k$：第$i$个特征的隐向量

### 复杂度优化

直接计算二阶项复杂度为 $O(kn^2)$，通过数学变换可优化到 $O(kn)$：

$$\sum_{i=1}^n \sum_{j=i+1}^n v_i^T v_j \cdot x_i x_j = \frac{1}{2} \sum_{f=1}^k \left[ \left( \sum_{i=1}^n v_{if} x_i \right)^2 - \sum_{i=1}^n v_{if}^2 x_i^2 \right]$$

### 时间复杂度对比

| 模型 | 训练复杂度 | 预测复杂度 |
|------|------------|------------|
| 多项式 | $O(n^2)$ | $O(n^2)$ |
| FM | $O(kn)$ | $O(kn)$ |

### FM 特点

| 优点 | 说明 |
|------|------|
| 自动特征交叉 | 自动学习所有二阶特征组合 |
| 参数高效 | 参数数量 $O(kn)$，可处理高维稀疏数据 |
| 泛化能力强 | 未出现的特征组合可通过隐向量内积计算 |
| 适用性广 | 可用于回归、分类、排序等任务 |

## 场感知因子分解机 (FFM)

### 核心思想

每个特征对每个场（field）学习不同的隐向量。

### FFM 公式

$$\hat{y} = w_0 + \sum_{i=1}^n w_i x_i + \sum_{i=1}^n \sum_{j=i+1}^n v_{i,f_j}^T v_{j,f_i} \cdot x_i x_j$$

其中：
- $f_i$：特征$i$所属的场
- $v_{i,f_j}$：特征$i$对场$f_j$的隐向量

### FM vs FFM

| 维度 | FM | FFM |
|------|-----|-----|
| 隐向量 | 每个特征一个隐向量 | 每个特征对每个场一个隐向量 |
| 参数数量 | $O(kn)$ | $O(knF)$，$F$为场数 |
| 表达能力 | 较弱 | 更强，考虑场信息 |
| 训练速度 | 快 | 慢（约慢10倍） |
| 适用场景 | 特征数多、场数少 | 场数多、特征间关系复杂 |

### 场的定义

```
示例：广告点击预测

特征：
- Publisher: ESPN, Vogue, NBC
- Advertiser: Nike, Gucci, Adidas
- Gender: Male, Female

场划分：
- Publisher 场：ESPN, Vogue, NBC
- Advertiser 场：Nike, Gucci, Adidas
- Gender 场：Male, Female

FFM 隐向量：
- ESPN 对 Advertiser 场的隐向量
- ESPN 对 Gender 场的隐向量
- Nike 对 Publisher 场的隐向量
- ...
```

## 深度因子分解机 (DeepFM)

### 架构

DeepFM 结合了 FM 的浅层特征交叉能力和 DNN 的深层特征学习能力：

```
输入特征
    ↓
┌─────────────────────────────────────┐
│              FM 部分                 │
│  一阶项 + 二阶交叉项（内积）          │
└─────────────────────────────────────┘
          ↓
┌─────────────────────────────────────┐
│              DNN 部分                │
│  Embedding → 多层全连接网络          │
└─────────────────────────────────────┘
          ↓
      FM输出 + DNN输出 → Sigmoid
```

### 公式

$$\hat{y} = \sigma(y_{FM} + y_{DNN})$$

其中：
$$y_{FM} = w_0 + \sum_{i=1}^n w_i x_i + \sum_{i=1}^n \sum_{j=i+1}^n v_i^T v_j \cdot x_i x_j$$

$$y_{DNN} = W^{|L|+1} \cdot a^L + b^{|L|+1}$$

### 特点

- **端到端训练**：FM 和 DNN 共享 Embedding
- **低阶+高阶**：同时学习低阶和高阶特征交叉
- **无需特征工程**：自动学习特征交叉

## 模型对比总结

| 模型 | 核心思想 | 特征交叉 | 适用场景 | 复杂度 |
|------|----------|----------|----------|--------|
| **SVD** | 矩阵分解 | 无 | 评分预测 | 低 |
| **ALS** | 交替优化 | 无 | 大规模评分预测 | 中 |
| **FM** | 隐向量内积 | 二阶自动交叉 | CTR预估、排序 | 低 |
| **FFM** | 场感知隐向量 | 二阶场感知交叉 | 场信息重要时 | 高 |
| **DeepFM** | FM + DNN | 低阶+高阶交叉 | 复杂特征交互 | 高 |

## 工业实践要点

### 1. 隐向量维度选择

| 数据规模 | 推荐维度 | 说明 |
|----------|----------|------|
| 小数据集 | 10-20 | 避免过拟合 |
| 中等数据集 | 50-100 | 平衡表达能力和泛化 |
| 大数据集 | 100-500 | 充分学习复杂模式 |

### 2. 正则化策略

- **L2正则化**：最常用，控制参数大小
- **Dropout**：DNN部分使用
- **Early Stopping**：防止过拟合

### 3. 负采样

对于隐式反馈数据（如点击），需要负采样：

```python
# 负采样示例
for user in users:
    # 正样本：用户交互过的物品
    pos_items = user_interactions[user]

    # 负样本：随机采样未交互的物品
    neg_items = random.sample(all_items - pos_items, k=len(pos_items))
```
