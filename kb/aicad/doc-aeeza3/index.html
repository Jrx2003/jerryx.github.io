常见问题

Q: 如何开始学习？

A: 建议从 Agent 基础概念 开始，然后依次阅读 ACI 概念与迁移 和 CadQuery 基础知识。

Q: MCP 和 ACI 的关系？

A: MCP 是通信协议，ACI 是设计范式。ACI 可以通过 MCP 实现。详见 CAD Sandbox MCP 设计详解 和 ACI 概念与迁移。

Q: 如何设计一个好的 CAD CLI 工具？

A: 参考 面向 LLM 的 CAD CLI 工具设计，遵循自描述、结构化输出、交互确认等原则。

Q: LLM 如何评估 CAD 模型质量？

A: 使用 LLM Judge 系统，从需求符合度、几何合理性、可制造性等维度评估。详见 借助 LLM 的评价系统设计。

Q: 如何优化渲染性能？

A: 参考 渲染与 Preview 模块设计，使用缓存、增量更新、批量渲染等策略。

Q: Few-Shot Learning 如何应用？

A: 参考 Prompt 设计 中的 Few-Shot 学习设计部分，根据任务类型选择合适的示例。
