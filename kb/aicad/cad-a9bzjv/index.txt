c:T1800,# CAD 动作集设计

> 更新日期：2026-03-12
> 来源：`docs/work_logs/WORK_LOG_2026_03_05_CAD_ACTION_INTERFACE.md`

## 概述

CAD 动作集是 ACI 的核心组成部分，将复杂的 CAD 操作拆解为基础动作单元。当前实现共 **15 个动作**，覆盖草图、实体、特征和状态操作。

---

## 动作分类

### 1. 草图操作 (Sketch Operations)

#### create_sketch

创建新的草图工作平面。

```json
{
  "action": "create_sketch",
  "plane": "XY"
}
```

**参数：**
- `plane`: 工作平面 ("XY", "XZ", "YZ")

---

#### add_rectangle

添加矩形到当前草图。

```json
{
  "action": "add_rectangle",
  "width": 80,
  "height": 40,
  "centered": true
}
```

**参数：**
- `width`: 宽度 (mm)
- `height`: 高度 (mm)
- `centered`: 是否居中 (默认 false)

---

#### add_circle

添加圆形到当前草图。

```json
{
  "action": "add_circle",
  "radius": 25
}
```

**参数：**
- `radius`: 半径 (mm)

---

#### add_polygon

添加多边形到当前草图。

```json
{
  "action": "add_polygon",
  "points": [[0,0], [50,0], [50,50], [0,50]]
}
```

**参数：**
- `points`: 顶点坐标列表 [[x,y], ...]

---

### 2. 实体操作 (Solid Operations)

#### extrude

拉伸草图创建实体。

```json
{
  "action": "extrude",
  "distance": 20,
  "direction": "up"
}
```

**参数：**
- `distance`: 拉伸距离 (mm)
- `direction`: 方向 ("up", "down")

---

#### cut_extrude

拉伸切除实体。

```json
{
  "action": "cut_extrude",
  "distance": 5,
  "target": "last_feature"
}
```

**参数：**
- `distance`: 切除距离 (mm)
- `target`: 目标特征 ("last_feature", "all", 或具体特征名)

---

#### revolve

旋转草图创建实体。

```json
{
  "action": "revolve",
  "axis": "X",
  "angle": 360
}
```

**参数：**
- `axis`: 旋转轴 ("X", "Y", "Z")
- `angle`: 旋转角度 (度)

---

#### loft

在多个剖面间创建放样实体。

```json
{
  "action": "loft",
  "profiles": ["sketch1", "sketch2"]
}
```

**参数：**
- `profiles`: 剖面草图名称列表

---

### 3. 特征操作 (Feature Operations)

#### fillet

添加圆角。

```json
{
  "action": "fillet",
  "radius": 2,
  "target": "all_outer_edges"
}
```

**参数：**
- `radius`: 圆角半径 (mm)
- `target`: 目标 ("all_outer_edges", "all_edges", "last_feature", "top_face" 等)

---

#### chamfer

添加倒角。

```json
{
  "action": "chamfer",
  "distance": 1,
  "target": "top_face"
}
```

**参数：**
- `distance`: 倒角距离 (mm)
- `target`: 目标面或边

---

#### hole

创建孔。

```json
{
  "action": "hole",
  "diameter": 5,
  "depth": 10,
  "position": [20, 20]
}
```

**参数：**
- `diameter`: 孔径 (mm)
- `depth`: 深度 (mm)
- `position`: 位置 [x, y]

---

#### pattern_linear

创建线性阵列。

```json
{
  "action": "pattern_linear",
  "feature": "hole_1",
  "count": 4,
  "spacing": 30,
  "direction": "X"
}
```

**参数：**
- `feature`: 要阵列的特征名称
- `count`: 数量
- `spacing`: 间距 (mm)
- `direction`: 方向 ("X", "Y")

---

#### pattern_circular

创建圆形阵列。

```json
{
  "action": "pattern_circular",
  "feature": "hole_1",
  "count": 6,
  "radius": 25
}
```

**参数：**
- `feature`: 要阵列的特征名称
- `count`: 数量
- `radius`: 圆周半径 (mm)

---

### 4. 状态操作 (State Operations)

#### rollback

回退到指定步骤。

```json
{
  "action": "rollback",
  "step": 2
}
```

**参数：**
- `step`: 要回退到的步骤编号

---

#### snapshot

创建当前状态的快照（用于分析或保存）。

```json
{
  "action": "snapshot"
}
```

---

#### modify_action (Phase B)

修改之前动作的参数。

```json
{
  "action": "modify_action",
  "target_step": 3,
  "new_params": {
    "radius": 3,
    "target": "all_outer_edges"
  }
}
```

**参数：**
- `target_step`: 要修改的动作步骤
- `new_params`: 新的参数字典

---

#### clear_session (Phase B)

清除当前会话，重新开始。

```json
{
  "action": "clear_session"
}
```

---

#### get_history (Phase B)

获取完整的动作历史。

```json
{
  "action": "get_history",
  "session_id": "my_model"
}
```

**参数：**
- `session_id`: 会话标识符

---

## 动作统计

| 类别 | 动作数 | 动作列表 |
|------|---------|----------|
| 草图操作 | 4 | create_sketch, add_rectangle, add_circle, add_polygon |
| 实体操作 | 4 | extrude, cut_extrude, revolve, loft |
| 特征操作 | 5 | fillet, chamfer, hole, pattern_linear, pattern_circular |
| 状态操作 | 2 | rollback, snapshot |
| **扩展操作** | 3 | modify_action, clear_session, get_history |
| **总计** | **18** | - |

---

## 目标选择器 (Target Selectors)

用于指定操作目标的选择器：

| 选择器 | 说明 |
|--------|------|
| `last_feature` | 最后创建的特征 |
| `all_outer_edges` | 所有外边 |
| `all_edges` | 所有边 |
| `all_faces` | 所有面 |
| `top_face` | 顶面（Z+ 方向）|
| `bottom_face` | 底面（Z- 方向）|
| `front_face` | 前面（Y+ 方向）|
| `back_face` | 后面（Y- 方向）|
| `left_face` | 左面（X- 方向）|
| `right_face` | 右面（X+ 方向）|

- `new_params`: 新的参数字典

---

#### clear_session (Phase B)

清除当前会话，重新开始。

```json
{
  "action": "clear_session"
}
```

#### get_history (Phase B)

获取完整的动作历史。

```json
{
  "action": "get_history",
  "session_id": "my_model"
}
```

**参数：**
- `session_id`: 会话标识符

| `all_outer_edges` | 所有外边 |

| `all_faces` | 所有面 |

| `bottom_face` | 底面（Z- 方向）|
| `front_face` | 前面（Y+ 方向）|
| `back_face` | 后面（Y- 方向）|

| `right_face` | 右面（X+ 方向）|
