核心术语

术语英文定义
Agent-LLM 驱动的智能体，负责特定任务的推理和决策
Sandbox-隔离执行环境，运行 CadQuery 代码并回收产物
MCPModel Context ProtocolLLM 与工具间的标准化通信协议
LLM Judge-使用 LLM 对生成结果进行质量评估的机制
ACIAgent-Computer InterfaceLLM 与 CAD 环境的交互接口
Action-基础 CAD 操作单元（如 extrude, fillet, hole）
Iteration-多轮优化的每一轮，包含观察、决策、执行
Rubric-评分标准和规则的配置文件
Evidence-评估所需的上下文信息（代码、产物、截图等）
GTGround Truth预先准备的标准答案，用作评估基准
Session-ACI 中的会话状态，跨多个 MCP 调用保持建模状态
Snapshot-动作执行后的状态快照，包含几何信息和多视图截图
