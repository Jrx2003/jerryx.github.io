## CAD 基础概念

### 什么是 CAD？

**CAD (Computer-Aided Design，计算机辅助设计)** 是使用计算机技术辅助进行设计、创建、分析和优化的技术。

---

### CAD 的主要类型

| 类型 | 描述 | 典型软件 |
|------|------|----------|
| 2D CAD | 二维绘图、平面设计 | AutoCAD 2D、DraftSight |
| 3D CAD | 三维建模、实体设计 | SolidWorks、Catia、Fusion 360 |
| 参数化 CAD | 基于参数的设计，修改参数自动更新模型 | CadQuery、OpenSCAD |
| 曲面建模 | NURBS 曲面建模 | Rhino、Alias |
| BIM | 建筑信息模型，含建筑全生命周期数据 | Revit、ArchiCAD |

---

### CAD 的核心工作流程

```
设计概念 → 参数化建模 → 仿真分析 → 工程图生成 → 数控加工
    ↓           ↓           ↓         ↓            ↓
  草图    →   3D模型   →   有限元  →   技术文档  →   G代码
```

---

## CadQuery 简介

### 什么是 CadQuery？

**CadQuery** 是一个基于 Python 的参数化 3D CAD 建模系统，它结合了 Python 的编程能力和 CAD 的建模能力。

---

### CadQuery 的特点

| 特点 | 说明 |
|------|------|
| **开源** | 完全免费，可扩展 |
| **Python 原生** | 利用 Python 生态系统 |
| **参数化** | 通过代码参数驱动模型 |
| **可编程** | 编写脚本自动生成模型 |
| **可组合** | 操作可以链式调用 |
| **支持多种输出格式** | STEP、STL、AMF 等 |

---

### CadQuery 与传统 CAD 的区别

| 维度 | 传统 CAD | CadQuery |
|------|----------|----------|
| **交互方式** | 图形界面（GUI） | 代码编程（CLI） |
| **设计方式** | 鼠标操作 | 编写脚本 |
| **可复用性** | 手动复制 | 代码复用 |
| **版本控制** | 文件版本 | Git 版本控制 |
| **自动化** | 宏/脚本 | 完全编程自动化 |
| **学习曲线** | 熟悉 GUI 需要时间 | 需要学习 Python |

## CadQuery 核心概念

### 1. Workplane (工作平面)

**定义：** CadQuery 的基础平面，用于进行 2D 草图操作。

**基本方向：**
- `XY` - 水平平面
- `XZ` - 侧向平面
- `YZ` - 垂直平面
- `XY` (反向) - 水平平面（Z 向下）

**创建方式：**
```python
import cadquery as cq

# 创建 XY 平面
wp = cq.Workplane('XY')

# 创建位于 Z=10 位置的 XY 平面
wp = cq.Workplane('XY').workplane(offsetZ=10)

# 创建选中的面作为工作平面
wp = cq.Workplane('XY').workplane(offsetZ=10)
```

---

### 2. Vector (向量/方向)

**定义：** 表示 3D 空间中的方向或位置。

**常用向量：**
```python
# 原点（原点）
(0, 0, 0)

# X 方向向量
(1, 0, 0)

# 上方向向量
(0, 0, 1)

# 通过点创建
v1 = cq.Vector(0, 1, 0)
v2 = cq.Vector(1, 1, 0)
```

**向量操作：**
```python
v1 + v2    # 向量加法
v1 - v2    # 向量减法
v1 * 2     # 标量乘法
v1.normalized()  # 归一化（单位向量）
v1.cross(v2)  # 叉积
v1.dot(v2)    # 点积
```

---

### 3. Shape (形状/几何体)

**定义：** CadQuery 中的几何对象。

**基本形状：**
- `Box` - 长方体
- `Cylinder` - 圆柱
- `Sphere` - 球体
- `Cone` - 圆锥

**创建示例：**
```python
import cadquery as cq

# 创建中心在原点的长方体（80x40x5）
box = cq.Workplane('XY').box(80, 40, 5, centered=(True, True, False))

# 创建圆柱（半径 10，高度 50）
cylinder = cq.Workplane('XY').circle(10).extrude(50)

# 创建球体（半径 15）
sphere = cq.Workplane('XY').sphere(15)
```

---

### 4. Wire (线框/轮廓)

**定义：** 2D 轮廓对象，用于创建 3D 实体。

**常用 Wire 操作：**
```python
import cadquery as cq

wp = cq.Workplane('XY')

# 创建矩形
rect = wp.rect(80, 40, forConstruction=False)

# 创建圆形
circle = wp.circle(25)

# 创建多边形
poly = wp.polygon([(0, 0), (50, 0), (50, 50), (0, 50)])

# 合并轮廓
combined = rect.union(circle)
```

---

### 5. Face (面)

**定义：** 3D 实体的表面。

**面的属性：**
- `normalAt()` - 获取某点的法向量
- `Center()` - 获取面的中心点
- `Area()` - 获取面的面积

**选择面的方式：**
```python
import cadquery as cq

# 创建长方体
box = cq.Workplane('XY').box(80, 40, 5)

# 选择上方的所有面
top_faces = box.faces('>Z')

# 选择 Z 方向最高的面
top_face = box.faces('>Z').faces('>Z')

# 获取面的中心
center = top_face.Center()
```

---

### 6. Edge (边)

**定义：** 两个面的交线。

**边的属性：**
- `Center()` - 获取边的中心点
- `Length()` - 获取边的长度

**选择边的方式：**
```python
import cadquery as cq

# 创建长方体
box = cq.Workplane('XY').box(80, 40, 5)

# 选择所有的边
all_edges = box.edges()

# 选择 Z 方向的外边
outer_edges = box.edges('>Z')

# 选择特定方向的边（如平行于 X 轴的边）
edges_parallel_to_x = box.edges('|X')
```

## CadQuery API 详解

### 链式调用 (Chaining)

CadQuery 的核心概念是链式调用，每个操作返回新的 Workplane 或 Shape 对象。

```python
import cadquery as cq

# 链式调用示例
result = (
    cq.Workplane('XY')
    .rect(80, 40)                     # 1. 创建矩形
    .extrude(5)                        # 2. 拉伸
    .faces('>Z')                       # 3. 选择上方面
    .workplane()                        # 4. 创建新的工作平面
    .rect(10, 10)                     # 5. 创建小矩形
    .cutBlind(-5)                       # 6. 切除
)
```

**链式调用的优势：**
- 代码可读性强
- 中间状态自然传递
- 避免重复选择

---

### 常用 Workplane 方法

#### workplane(offsetX, offsetZ)

```python
# 在 Z 方向偏移 10mm
wp = cq.Workplane('XY').workplane(offsetZ=10)
```

#### rotated(degrees)

```python
# 绕 Z 轴旋转 45 度
wp = cq.Workplane('XY').rotated((0, 0, 1), 45)
```

---

### 常用 Shape 方法

#### fillet(radius, edges=None)

```python
# 创建带圆角的长方体
result = (
    cq.Workplane('XY')
    .box(80, 40, 5)
    .edges('>Z').fillet(2)  # 对所有上边进行 2mm 圆角
)
```

#### chamfer(length, length2=None, edges=None)

```python
# 创建带倒角的长方体
result = (
    cq.Workplane('XY')
    .box(80, 40, 5)
    .edges('>Z').chamfer(1)  # 对所有上边进行 1mm 倒角
)
```

#### hole(diameter, depth=None, clean=True)

```python
# 在当前面打孔
result = (
    cq.Workplane('XY')
    .rect(80, 40)
    .extrude(5)
    .faces('>Z')
    .workplane()
    .hole(5)  # 打一个直径 5mm 的通孔
)

# 打多个孔
result = (
    cq.Workplane('XY')
    .rect(80, 40)
    .extrude(5)
    .faces('>Z')
    .workplane()
    .pushPoints([(20, 15), (-20, 15), (-20, -15), (20, -15)])
    .hole(5)  # 在四个位置各打一个孔
)
```

## CadQuery 常用操作

### 1. 创建基本实体

#### 长方体

```python
import cadquery as cq

# 方式 1：以原点为中心
box1 = cq.Workplane('XY').box(80, 40, 5)

# 方式 2：指定中心点
box2 = cq.Workplane('XY').box(80, 40, 5, centered=(True, True, False))

# 方式 3：指定底面中心，向 Z 正方向拉伸
box3 = cq.Workplane('XY').box(80, 40, 5, centered=(True, True, False))
```

#### 圆柱

```python
# 基于半径拉伸圆形
cylinder1 = cq.Workplane('XY').circle(25).extrude(100)

# 基于高度创建圆柱
cylinder2 = cq.Workplane('XY').cylinder(height=100, radius=25)
```

#### 管道/空心圆柱

```python
# 通过差集创建空心圆柱
outer_cylinder = cq.Workplane('XY').circle(25).extrude(100)
inner_cylinder = cq.Workplane('XY').circle(20).extrude(100)
pipe = outer_cylinder.cut(inner_cylinder)

# 或者使用 hollow 函数
pipe2 = cq.Workplane('XY').circle(25).hollow(height=100, wallThickness=5)
```

---

### 2. 创建复杂形状

#### 多边形拉伸

```python
# 创建六边形
hexagon = cq.Workplane('XY').polygon([
    (50, 0),
    (25, 43.3),
    (-25, 43.3),
    (-50, 0),
    (-25, -43.3),
    (25, -43.3)
])

result = hexagon.extrude(20)
```

#### 放样（Loft）

```python
import cadquery as cq

# 底部轮廓
bottom = cq.Workplane('XY').circle(20)

# 顶部轮廓
top = cq.Workplane('XY').workplane(offsetZ=50).circle(30)

# 放样创建实体
lofted = bottom.loft(top)
```

#### 旋转（Revolve）

```python
# 创建轮廓
profile = cq.Workplane('XZ').polyline([(0, 0), (30, 10), (20, 30), (10, 10)])

# 绕 X 轴旋转 360 度
result = profile.revolve(360)
```

---

### 3. 特征操作

#### 阵列

```python
# 矩形阵列
rect = cq.Workplane('XY').rect(20, 20)

# 沿 X 方向创建 3 个阵列
result = (
    cq.Workplane('XY')
    .rect(20, 20)
    .workplane()
    .rectArray(10, 20, xSpacing=30, yCount=1, ySpacing=0)
)
```

#### 环形阵列

```python
# 创建圆形阵列
result = (
    cq.Workplane('XY')
    .workplane(offsetZ=0)
    .circle(30)
    .polarArray(6, 50)  # 6 个元素，半径 50mm
    .extrude(10)
)
```

#### 镜像

```python
# 创建形状
shape = cq.Workplane('XY').rect(20, 20).extrude(10)

# 沿 X 轴镜像
result = shape.mirror(mirrorPlane='XY')

# 沿 XZ 平面镜像
result = shape.mirror(mirrorPlane='XZ')
```

#### 缩放

```python
shape = cq.Workplane('XY').box(40, 40, 20)

# 缩放到 1.5 倍
scaled = shape.scale(1.5)

# 缩放到特定尺寸
scaled2 = shape.scale((1.0, 1.5, 1.0))  # Y 方向缩放 1.5 倍
```

## CadQuery 高级主题

### 1. 参数化设计

```python
import cadquery as cq

# 定义参数
length = 80
width = 40
thickness = 5
hole_radius = 5
hole_positions = [(20, 15), (-20, 15), (-20, -15), (20, -15)]

# 参数化创建
result = (
    cq.Workplane('XY')
    .rect(length, width)
    .extrude(thickness)
    .faces('>Z')
    .workplane()
    .pushPoints(hole_positions)
    .hole(hole_radius)
)
```

---

### 2. 约束和参考

```python
import cadquery as cq

# 创建参考几何

## CAD 基础概念

### 什么是 CAD？

**CAD (Computer-Aided Design，计算机辅助设计)** 是使用计算机技术辅助进行设计、创建、分析和优化的技术。

---

### CAD 的主要类型

| 类型 | 描述 | 典型软件 |
|------|------|----------|
| 2D CAD | 二维绘图、平面设计 | AutoCAD 2D、DraftSight |
| 3D CAD | 三维建模、实体设计 | SolidWorks、Catia、Fusion 360 |
| 参数化 CAD | 基于参数的设计，修改参数自动更新模型 | CadQuery、OpenSCAD |
| 曲面建模 | NURBS 曲面建模 | Rhino、Alias |
| BIM | 建筑信息模型，含建筑全生命周期数据 | Revit、ArchiCAD |

---

### CAD 的核心工作流程

```
设计概念 → 参数化建模 → 仿真分析 → 工程图生成 → 数控加工
    ↓           ↓           ↓         ↓            ↓
  草图    →   3D模型   →   有限元  →   技术文档  →   G代码
```

## CadQuery 简介

### 什么是 CadQuery？

**CadQuery** 是一个基于 Python 的参数化 3D CAD 建模系统，它结合了 Python 的编程能力和 CAD 的建模能力。

---

### CadQuery 的特点

| 特点 | 说明 |
|------|------|
| **开源** | 完全免费，可扩展 |
| **Python 原生** | 利用 Python 生态系统 |
| **参数化** | 通过代码参数驱动模型 |
| **可编程** | 编写脚本自动生成模型 |
| **可组合** | 操作可以链式调用 |
| **支持多种输出格式** | STEP、STL、AMF 等 |

---

### CadQuery 与传统 CAD 的区别

| 维度 | 传统 CAD | CadQuery |
|------|----------|----------|
| **交互方式** | 图形界面（GUI） | 代码编程（CLI） |
| **设计方式** | 鼠标操作 | 编写脚本 |
| **可复用性** | 手动复制 | 代码复用 |
| **版本控制** | 文件版本 | Git 版本控制 |
| **自动化** | 宏/脚本 | 完全编程自动化 |
| **学习曲线** | 熟悉 GUI 需要时间 | 需要学习 Python |

## CadQuery 核心概念

### 1. Workplane (工作平面)

**定义：** CadQuery 的基础平面，用于进行 2D 草图操作。

**基本方向：**
- `XY` - 水平平面
- `XZ` - 侧向平面
- `YZ` - 垂直平面
- `XY` (反向) - 水平平面（Z 向下）

**创建方式：**
```python
import cadquery as cq

# 创建 XY 平面
wp = cq.Workplane('XY')

# 创建位于 Z=10 位置的 XY 平面
wp = cq.Workplane('XY').workplane(offsetZ=10)

# 创建选中的面作为工作平面
wp = cq.Workplane('XY').workplane(offsetZ=10)
```

---

### 2. Vector (向量/方向)

**定义：** 表示 3D 空间中的方向或位置。

**常用向量：**
```python
# 原点（原点）
(0, 0, 0)

# X 方向向量
(1, 0, 0)

# 上方向向量
(0, 0, 1)

# 通过点创建
v1 = cq.Vector(0, 1, 0)
v2 = cq.Vector(1, 1, 0)
```

**向量操作：**
```python
v1 + v2    # 向量加法
v1 - v2    # 向量减法
v1 * 2     # 标量乘法
v1.normalized()  # 归一化（单位向量）
v1.cross(v2)  # 叉积
v1.dot(v2)    # 点积
```

---

### 3. Shape (形状/几何体)

**定义：** CadQuery 中的几何对象。

**基本形状：**
- `Box` - 长方体
- `Cylinder` - 圆柱
- `Sphere` - 球体
- `Cone` - 圆锥

**创建示例：**
```python
import cadquery as cq

# 创建中心在原点的长方体（80x40x5）
box = cq.Workplane('XY').box(80, 40, 5, centered=(True, True, False))

# 创建圆柱（半径 10，高度 50）
cylinder = cq.Workplane('XY').circle(10).extrude(50)

# 创建球体（半径 15）
sphere = cq.Workplane('XY').sphere(15)
```

---

### 4. Wire (线框/轮廓)

**定义：** 2D 轮廓对象，用于创建 3D 实体。

**常用 Wire 操作：**
```python
import cadquery as cq

wp = cq.Workplane('XY')

# 创建矩形
rect = wp.rect(80, 40, forConstruction=False)

# 创建圆形
circle = wp.circle(25)

# 创建多边形
poly = wp.polygon([(0, 0), (50, 0), (50, 50), (0, 50)])

# 合并轮廓
combined = rect.union(circle)
```

---

### 5. Face (面)

**定义：** 3D 实体的表面。

**面的属性：**
- `normalAt()` - 获取某点的法向量
- `Center()` - 获取面的中心点
- `Area()` - 获取面的面积

**选择面的方式：**
```python
import cadquery as cq

# 创建长方体
box = cq.Workplane('XY').box(80, 40, 5)

# 选择上方的所有面
top_faces = box.faces('>Z')

# 选择 Z 方向最高的面
top_face = box.faces('>Z').faces('>Z')

# 获取面的中心
center = top_face.Center()
```

---

### 6. Edge (边)

**定义：** 两个面的交线。

**边的属性：**
- `Center()` - 获取边的中心点
- `Length()` - 获取边的长度

**选择边的方式：**
```python
import cadquery as cq

# 创建长方体
box = cq.Workplane('XY').box(80, 40, 5)

# 选择所有的边
all_edges = box.edges()

# 选择 Z 方向的外边
outer_edges = box.edges('>Z')

# 选择特定方向的边（如平行于 X 轴的边）
edges_parallel_to_x = box.edges('|X')
```

## CadQuery API 详解

### 链式调用 (Chaining)

CadQuery 的核心概念是链式调用，每个操作返回新的 Workplane 或 Shape 对象。

```python
import cadquery as cq

# 链式调用示例
result = (
    cq.Workplane('XY')
    .rect(80, 40)                     # 1. 创建矩形
    .extrude(5)                        # 2. 拉伸
    .faces('>Z')                       # 3. 选择上方面
    .workplane()                        # 4. 创建新的工作平面
    .rect(10, 10)                     # 5. 创建小矩形
    .cutBlind(-5)                       # 6. 切除
)
```

**链式调用的优势：**
- 代码可读性强
- 中间状态自然传递
- 避免重复选择

---

### 常用 Workplane 方法

#### workplane(offsetX, offsetZ)

```python
# 在 Z 方向偏移 10mm
wp = cq.Workplane('XY').workplane(offsetZ=10)
```

#### rotated(degrees)

```python
# 绕 Z 轴旋转 45 度
wp = cq.Workplane('XY').rotated((0, 0, 1), 45)
```

---

### 常用 Shape 方法

#### fillet(radius, edges=None)

```python
# 创建带圆角的长方体
result = (
    cq.Workplane('XY')
    .box(80, 40, 5)
    .edges('>Z').fillet(2)  # 对所有上边进行 2mm 圆角
)
```

#### chamfer(length, length2=None, edges=None)

```python
# 创建带倒角的长方体
result = (
    cq.Workplane('XY')
    .box(80, 40, 5)
    .edges('>Z').chamfer(1)  # 对所有上边进行 1mm 倒角
)
```

#### hole(diameter, depth=None, clean=True)

```python
# 在当前面打孔
result = (
    cq.Workplane('XY')
    .rect(80, 40)
    .extrude(5)
    .faces('>Z')
    .workplane()
    .hole(5)  # 打一个直径 5mm 的通孔
)

# 打多个孔
result = (
    cq.Workplane('XY')
    .rect(80, 40)
    .extrude(5)
    .faces('>Z')
    .workplane()
    .pushPoints([(20, 15), (-20, 15), (-20, -15), (20, -15)])
    .hole(5)  # 在四个位置各打一个孔
)
```

## CadQuery 常用操作

### 1. 创建基本实体

#### 长方体

```python
import cadquery as cq

# 方式 1：以原点为中心
box1 = cq.Workplane('XY').box(80, 40, 5)

box2 = cq.Workplane('XY').box(80, 40, 5, centered=(True, True, False))

```

#### 圆柱

```python
# 基于半径拉伸圆形
cylinder1 = cq.Workplane('XY').circle(25).extrude(100)

cylinder2 = cq.Workplane('XY').cylinder(height=100, radius=25)
```

#### 管道/空心圆柱

```python
# 通过差集创建空心圆柱
outer_cylinder = cq.Workplane('XY').circle(25).extrude(100)
inner_cylinder = cq.Workplane('XY').circle(20).extrude(100)
pipe = outer_cylinder.cut(inner_cylinder)

# 或者使用 hollow 函数
pipe2 = cq.Workplane('XY').circle(25).hollow(height=100, wallThickness=5)
```

---

### 2. 创建复杂形状

#### 多边形拉伸

```python
# 创建六边形
hexagon = cq.Workplane('XY').polygon([
    (50, 0),
    (25, 43.3),
    (-25, 43.3),
    (-50, 0),
    (-25, -43.3),
    (25, -43.3)
])

result = hexagon.extrude(20)
```

#### 放样（Loft）

```python
import cadquery as cq

# 底部轮廓
bottom = cq.Workplane('XY').circle(20)

# 顶部轮廓
top = cq.Workplane('XY').workplane(offsetZ=50).circle(30)

# 放样创建实体
lofted = bottom.loft(top)
```

#### 旋转（Revolve）

```python
# 创建轮廓
profile = cq.Workplane('XZ').polyline([(0, 0), (30, 10), (20, 30), (10, 10)])

# 绕 X 轴旋转 360 度
result = profile.revolve(360)
```

---

### 3. 特征操作

#### 阵列

```python
# 矩形阵列
rect = cq.Workplane('XY').rect(20, 20)

# 沿 X 方向创建 3 个阵列
result = (
    cq.Workplane('XY')
    .rect(20, 20)
    .workplane()

)
```

#### 环形阵列

```python
# 创建圆形阵列
result = (
    cq.Workplane('XY')
    .workplane(offsetZ=0)
    .circle(30)
    .polarArray(6, 50)  # 6 个元素，半径 50mm
    .extrude(10)
)
```

#### 镜像

```python
# 创建形状
shape = cq.Workplane('XY').rect(20, 20).extrude(10)

# 沿 X 轴镜像
result = shape.mirror(mirrorPlane='XY')

# 沿 XZ 平面镜像

```

#### 缩放

```python
shape = cq.Workplane('XY').box(40, 40, 20)

# 缩放到 1.5 倍
scaled = shape.scale(1.5)

# 缩放到特定尺寸
scaled2 = shape.scale((1.0, 1.5, 1.0))  # Y 方向缩放 1.5 倍
