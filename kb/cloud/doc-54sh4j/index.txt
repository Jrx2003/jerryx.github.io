## 主流云厂商

### 全球三大云

```
市场份额 (2024):

AWS: 31%
├── 最早进入，生态最完善
├── 200+ 服务
└── 全球区域最多

Azure: 24%
├── 微软生态集成
├── 企业客户优势
└── 混合云能力强

GCP: 11%
├── AI/ML 领先
├── Kubernetes 发源地
└── 数据分析和开源
```

### 国内云厂商

| 厂商 | 优势领域 | 特点 |
|------|----------|------|
| 阿里云 | 电商、金融 | 国内市场份额第一 |
| 腾讯云 | 游戏、社交 | 微信生态集成 |
| 华为云 | 政企、制造 | 硬件协同优势 |
| 百度云 | AI | 大模型服务 |

## 服务能力对比

### 核心服务对照表

| 服务类型 | AWS | Azure | 阿里云 |
|----------|-----|-------|--------|
| 计算 | EC2 | VM | ECS |
| 对象存储 | S3 | Blob | OSS |
| K8s | EKS | AKS | ACK |
| Serverless | Lambda | Functions | FC |
| 数据库 | RDS | SQL DB | RDS |
| CDN | CloudFront | CDN | CDN |
| 消息队列 | SQS/SNS | Service Bus | MNS |

### 差异化能力

```
AWS 优势:
- 服务种类最全
- 全球基础设施
- 企业级支持

Azure 优势:
- 混合云 (Azure Stack)
- Office 365 集成
- Windows 生态

GCP 优势:
- BigQuery 数据分析
- AI Platform
- 开源技术 (K8s, TensorFlow)

阿里云优势:
- 双11验证的技术
- 电商行业解决方案
- 国内合规认证
```

## 价格对比

### 定价模式

```
通用模式:
1. 按需 (On-Demand): 单价最高，最灵活
2. 预留 (Reserved): 承诺1-3年，折扣40-60%
3. 竞价 (Spot): 利用闲置资源，折扣60-90%

注意事项:
- 出口流量费用差异大
- 存储操作费用容易被忽略
- API 调用可能收费
```

### 价格对比示例

```
通用型虚拟机 (2vCPU 8GB) 按需价格对比:

┌──────────┬──────────┬────────────┐
│ 厂商     │ 价格/月  │ 备注       │
├──────────┼──────────┼────────────┤
│ AWS      │ ~$70     │ us-east-1  │
│ Azure    │ ~$70     │ East US    │
│ GCP      │ ~$65     │ us-central │
│ 阿里云   │ ~$45     │ 华东1      │
│ 腾讯云   │ ~$42     │ 广州       │
└──────────┴──────────┴────────────┘

*价格仅供参考，实际以官网为准
```

## 选型建议

### 选型维度

```
技术维度:
├── 服务完整度
├── 技术先进性
├── 文档和社区
└── 开发者体验

业务维度:
├── 合规要求 (数据主权)
├── 延迟要求 (区域选择)
├── 生态集成 (现有系统)
└── 成本预算

战略维度:
├── 厂商锁定风险
├── 多云策略
├── 供应商关系
└── 长期路线图
```

### 场景建议

```
初创公司:
- 推荐: AWS/Azure 免费额度
- 原因: 服务全、文档好、生态成熟

国内企业:
- 推荐: 阿里云/腾讯云
- 原因: 合规、客服、网络延迟

AI/ML 项目:
- 推荐: GCP 或 阿里云 PAI
- 原因: AI 平台完善

混合云需求:
- 推荐: Azure / 华为云
- 原因: 混合云方案成熟
```

## 面试要点

**Q1: 如何选择云厂商？**

A:
- 技术需求匹配度
- 成本结构分析
- 合规和安全要求
- 团队熟悉程度
- 生态集成需求

**Q2: 多云策略的优缺点？**

A:
- 优点: 避免锁定、容灾、选择最优服务
- 缺点: 复杂度增加、成本可能更高、人才要求

**Q3: 云迁移的主要考虑？**

A:
- 应用架构改造
- 数据迁移方案
- 网络连通性
- 成本对比分析
- 团队培训
