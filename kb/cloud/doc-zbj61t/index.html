八、面试要点

高频问题

  1. Serverless 冷启动如何优化？

    • SnapStart 预编译镜像
    • 预置并发保持热实例
    • 精简代码和依赖

  2. WASM 相比容器的优势？

    • 启动更快（毫秒级）
    • 占用更小（KB 级）
    • 跨平台兼容

  3. 边缘计算的应用场景？

    • IoT 设备
    • 实时分析
    • 低延迟应用（AR/VR）

  4. FinOps 的核心是什么？

    • 云成本可视化
    • 优化资源使用
    • 预测和规划

  5. 机密计算解决了什么问题？

    • 数据使用过程中的安全
    • 敏感数据处理
    • 多方计算
