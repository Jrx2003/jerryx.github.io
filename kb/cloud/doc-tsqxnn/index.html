й«ҳйў‘й—®йўҳ

  1. е®№еҷЁе’ҢиҷҡжӢҹжңәзҡ„ж №жң¬еҢәеҲ«пјҹ

    • иҷҡжӢҹжңәпјҡзЎ¬д»¶зә§иҷҡжӢҹеҢ–пјҢзӢ¬з«Ӣ OS
    • е®№еҷЁпјҡж“ҚдҪңзі»з»ҹзә§иҷҡжӢҹеҢ–пјҢе…ұдә«еҶ…ж ё

  2. Docker зҡ„ Namespace е’Ң Cgroups дҪңз”Ёпјҹ

    • Namespaceпјҡйҡ”зҰ»зҺҜеўғпјҲиҝӣзЁӢгҖҒзҪ‘з»ңгҖҒж–Үд»¶зі»з»ҹпјү
    • CgroupsпјҡйҷҗеҲ¶иө„жәҗпјҲCPUгҖҒеҶ…еӯҳпјү

  3. Kata зӣёжҜ” Docker зҡ„дјҳеҠҝпјҹ

    • зӢ¬з«ӢеҶ…ж ёпјҢжӣҙејәйҡ”зҰ»
    • йҖӮеҗҲеӨҡз§ҹжҲ·еңәжҷҜ
    • зүәзүІйғЁеҲҶжҖ§иғҪжҚўеҸ–е®үе…Ё

  4. еҰӮдҪ•йҖүжӢ©е®№еҷЁиҝҗиЎҢж—¶пјҹ

    • DockerпјҡйҖҡз”ЁеңәжҷҜпјҢиҝҪжұӮжҖ§иғҪ
    • KataпјҡеӨҡз§ҹжҲ·гҖҒй«ҳе®үе…ЁиҰҒжұӮ
    • gVisorпјҡдёӯзӯүйҡ”зҰ»пјҢжҖ§иғҪжҚҹиҖ—иҫғе°Ҹ

д№қгҖҒзӣёе…іжҠҖжңҜ

  • Kubernetesжһ¶жһ„иҜҰи§Ј - е®№еҷЁзј–жҺ’
  • и®Ўз®—жңҚеҠЎжҰӮи§Ҳ - дә‘и®Ўз®—жңҚеҠЎ
  • дә‘и®Ўз®—ж ёеҝғжҰӮеҝө - еҹәзЎҖжҰӮеҝө
