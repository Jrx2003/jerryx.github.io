# 风险管理

> 量化交易的核心——控制风险，保住本金

---

## 目录说明

本目录介绍量化交易中的风险管理方法。许多初学者只关注策略收益，而忽视了风险管理。事实上，风险管理是量化投资成功的关键——一个夏普比率 1.5 但风控完善的策略，往往比一个夏普比率 2.0 但风控缺失的策略更适合实盘。

## 风险管理框架

### 事前风险管理（Before Trade）
1. **策略风险评估**: 回测中的最大回撤、波动率
2. **仓位规划**: 单笔交易最大亏损、总仓位限制
3. **压力测试**: 在历史极端行情下的表现

### 事中风控（During Trade）
1. **实时监控**: 持仓盈亏、风险敞口、希腊字母（期权）
2. **预警系统**: 接近止损线时自动提醒
3. **自动止损**: 达到止损条件自动平仓

### 事后分析（After Trade）
1. **盈亏归因**: 收益来自哪里？风险来自哪里？
2. **策略健康度**: 策略是否还在正常运作？
3. **参数再评估**: 是否需要调整策略参数？

## 关键风险指标速查

| 指标 | 计算公式 | 健康范围 | 警戒线 |
|------|----------|----------|--------|
| 最大回撤 | (峰值-谷底)/峰值 | < 20% | > 30% |
| 夏普比率 | (年化收益-无风险利率)/年化波动 | > 1.0 | < 0.5 |
| 日VaR(95%) | 历史模拟或参数法 | < 账户2% | > 账户5% |
| 波动率 | 收益率标准差×√252 | < 15% | > 25% |
| 贝塔 | 组合收益与市场收益的协方差/市场方差 | 0.5-1.5 | <-0.5 或 >2 |

## 风险管理的常见误区

1. **只看收益不看风险**: 高收益往往伴随高风险
2. **止损设得太紧**: 频繁止损，累积亏损
3. **过度分散**: 分散到相关性高的资产无效
4. **忽视尾部风险**: 正态分布假设低估极端事件
5. **风险指标误用**: VaR单独使用不充分
