# 金融市场概述

> 理解量化交易的战场——金融市场的运作机制

---

## 什么是金融市场？

**金融市场**是买卖双方交易金融资产（股票、债券、期货等）的场所。

简单类比：
- **农贸市场**：买卖蔬菜、水果
- **金融市场**：买卖股票、债券、货币

### 金融市场的核心功能

1. **融资功能**：企业发行股票/债券筹集资金
2. **投资功能**：投资者获取收益
3. **价格发现**：供求关系决定资产价格
4. **风险管理**：通过衍生品对冲风险
5. **流动性提供**：随时可以买卖

---

## 金融市场的分类

### 按交易标的分类

| 市场类型 | 交易标的 | 例子 |
|---------|---------|------|
| **股票市场** | 股票（公司所有权） | A股、美股、港股 |
| **债券市场** | 债券（债权债务关系） | 国债、企业债 |
| **外汇市场** | 货币 | 美元兑人民币 |
| **衍生品市场** | 期货、期权、互换 | 股指期货、商品期货 |
| **大宗商品市场** | 实物商品 | 原油、黄金、大豆 |

### 按交易场所分类

#### 1. 交易所市场（Exchange）
- **特点**：集中交易、标准化合约、透明
- **例子**：上海证券交易所、纽约证券交易所(NYSE)
- **优点**：流动性好、监管严格、公平透明

#### 2. 场外市场（OTC, Over-the-Counter）
- **特点**：双方直接协商、非标准化
- **例子**：银行间外汇市场、OTC衍生品
- **优点**：灵活定制、隐私性好

### 按交割时间分类

| 市场类型 | 交割时间 | 特点 |
|---------|---------|------|
| **现货市场** | 即时或T+1交割 | 实际资产所有权转移 |
| **期货市场** | 未来约定日期交割 | 杠杆交易、对冲工具 |

## 市场参与者

### 1. 个人投资者（散户）
- **特点**：资金量小、信息获取能力弱
- **目标**：资产增值、跑赢通胀
- **劣势**：容易情绪化、缺乏专业分析能力

### 2. 机构投资者

#### 公募基金
- **特点**：向社会公众募集资金
- **监管**：严格，信息披露要求高
- **目标**：相对收益（跑赢基准）

#### 私募基金/对冲基金
- **特点**：向特定投资者募集，策略灵活
- **策略**：量化、宏观、事件驱动等多种策略
- **目标**：绝对收益（无论市场涨跌都要赚钱）

#### 保险公司/养老金
- **特点**：资金规模大、投资期限长
- **目标**：稳健收益、资产负债匹配

### 3. 做市商（Market Makers）
- **作用**：提供买卖双边报价，维持流动性
- **收益**：买卖价差（Bid-Ask Spread）
- **例子**：高频交易公司、投资银行交易部门

### 4. 高频交易者（HFT）
- **特点**：极短时间持仓、大量交易
- **策略**：延迟套利、做市、统计套利
- **技术**：超低延迟系统、共置服务器

## 订单类型

### 1. 市价单（Market Order）
**定义**：以当前最优价格立即成交

**优点**：
- 保证成交
- 适合紧急交易

**缺点**：
- 价格不确定（可能滑点）
- 大额订单冲击成本高

**适用场景**：需要立即执行，对价格不敏感

### 2. 限价单（Limit Order）
**定义**：指定价格，只有达到该价格才成交

**优点**：
- 控制成交价格
- 可以挂低价买/高价卖

**缺点**：
- 不保证成交
- 可能错过机会

**适用场景**：对价格有明确要求，可以等待

### 3. 止损单（Stop Order/Stop-Loss）
**定义**：当价格达到触发价时，转为市价单

**用途**：
- 控制亏损（跌破某价位自动卖出）
- 突破交易（突破某价位自动买入）

**注意**：
- 触发≠成交价格（特别是快速下跌时）
- 可能有滑点

### 4. 止损限价单（Stop-Limit Order）
**定义**：触发后变为限价单而非市价单

**优点**：控制触发后的成交价格
**缺点**：可能无法成交

## 交易机制

### 连续竞价 vs 集合竞价

#### 集合竞价（Call Auction）
- **时间**：开盘前、收盘前
- **机制**：收集所有订单，统一撮合
- **价格**：使成交量最大的价格
- **目的**：确定开盘价、收盘价，防止操纵

#### 连续竞价（Continuous Trading）
- **时间**：大部分交易时间
- **机制**：价格优先、时间优先
- **成交**：有对手盘就立即成交

### 价格优先、时间优先原则

**例子**：
```
买单队列：
1. 买100股 @ 10.05（时间9:30:01）
2. 买50股 @ 10.05（时间9:30:02）
3. 买200股 @ 10.03（时间9:30:00）

卖单来了：卖150股 @ 10.05

成交结果：
- 先与买单1成交100股
- 再与买单2成交50股
- 买单3价格不够，无法成交
```

## 交易成本

### 显性成本
1. **佣金**：券商收取的交易费用
2. **印花税**：卖出时缴纳（A股）
3. **过户费**：股票过户费用

### 隐性成本（更重要！）
1. **买卖价差（Bid-Ask Spread）**
   - 买一价 vs 卖一价的差额
   - 流动性越差，价差越大

2. **滑点（Slippage）**
   - 下单价格 vs 实际成交价格的差异
   - 市场波动大、订单大时更严重

3. **市场冲击（Market Impact）**
   - 大额订单推动价格向不利方向移动
   - 买入时推高价格，卖出时压低价格

**成本举例**：
```
假设买入1000股，股价10元

显性成本：
- 佣金：10元（万1）
- 印花税：0（买入不收）
- 过户费：0.1元

隐性成本：
- 滑点：10.00 → 10.02（买入推高）
- 额外成本：1000 × 0.02 = 20元

总成本：30.1元 = 0.3% 的交易金额
```

## 市场微观结构

### 订单簿（Order Book）
显示当前所有未成交订单的列表。

**例子**：
```
Level-2行情

卖五: 10.06  500股
卖四: 10.05  800股
卖三: 10.04  1200股
卖二: 10.03  2000股
卖一: 10.02  3000股  ← 当前卖价 (Ask)
---------------------------------
买一: 10.01  2500股  ← 当前买价 (Bid)
买二: 10.00  1500股
买三:  9.99  1000股
买四:  9.98   800股
买五:  9.97   600股

价差 = 10.02 - 10.01 = 0.01 (0.1%)
```

### 流动性（Liquidity）

**定义**：能够以合理价格快速买卖的能力。

**度量**：
1. **买卖价差**：越小流动性越好
2. **订单簿深度**：买卖挂单量
3. **成交量**：日均成交量

**重要性**：
- 流动性差 → 滑点大 → 策略失效
- 量化策略通常选择流动性好的股票

### 波动性（Volatility）

**定义**：价格变动的剧烈程度。

**度量**：
- 历史波动率：收益率的标准差
- 隐含波动率：期权价格反推的波动率

**与量化交易的关系**：
- 高波动 → 机会多，但风险也大
- 低波动 → 机会少，但更稳定

## A股市场特点

### 交易规则

| 项目 | A股规则 |
|------|---------|
| **交易时间** | 9:30-11:30, 13:00-15:00 |
| **T+1制度** | 当天买入的股票，次日才能卖出 |
| **涨跌停板** | ±10%（ST股±5%，创业板/科创板±20%） |
| **最小单位** | 100股（1手） |
| **印花税** | 卖出时0.05% |

### 市场分层

1. **主板**：大盘股（600xxx, 000xxx）
2. **中小板**：已并入主板
3. **创业板**：创新成长企业（300xxx）
4. **科创板**：硬科技企业（688xxx）
5. **北交所**：中小企业

### 特殊机制

- **融资融券**：可以借钱买股票（融资），或借股票卖出（融券）
- **沪港通/深港通**：境外投资者投资A股的渠道

## 美股市场特点

### 主要交易所
- **NYSE**：纽约证券交易所（传统大公司）
- **NASDAQ**：纳斯达克（科技股为主）

### 交易规则

| 项目 | 美股规则 |
|------|---------|
| **T+0制度** | 当天可以多次买卖 |
| **无涨跌停** | 理论上可以无限涨跌 |
| **盘后交易** | 正常交易时间外可交易 |
| **最小单位** | 1股 |

### 做空机制
- 可以借股票卖出，期待价格下跌后买回归还
- 有专门的做空机构（如浑水）

## 对量化交易的影响

### 市场特征与策略选择

| 市场特征 | 适合的策略 | 注意事项 |
|---------|-----------|---------|
| 高流动性 | 高频、统计套利 | 竞争激烈 |
| 低流动性 | 低频、价值投资 | 冲击成本大 |
| 高波动 | 动量、趋势跟踪 | 风险管理 |
| 低波动 | 均值回归、套利 | 机会少 |

### 规则限制与对策

**A股T+1限制**：
- 影响：日内策略无法当日平仓
- 对策：使用ETF融券、股指期货对冲

**涨跌停限制**：
- 影响：流动性枯竭、无法成交
- 对策：避开小盘股、设置更宽的止损
