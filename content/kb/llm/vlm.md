## VLM 概述

### 什么是视觉语言模型

能够同时理解图像和文本，进行多模态推理的模型。

```
能力:
- 图像描述
- 视觉问答
- 图文检索
- 多模态对话
```

## 架构设计

### LLaVA 架构

```
图像 → ViT → 投影层 → LLM
                          ↑
文本 → Tokenizer → Embedding
```

### 训练流程

```
1. 预训练: 对齐视觉和文本
2. 指令微调: 学习多模态指令
```
