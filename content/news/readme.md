# News - 科技与 AI 新闻简报

本目录存放每日科技与 AI 领域的热点新闻简报。

## 目录说明

每日使用 `/每日热点` skill 生成的简报将存放在按年月组织的子目录中，文件命名格式为 `YYYY/MM/DD热点.md`（如：`News/2026/03/13热点.md`）。

## 简报内容

每份简报包含以下内容：

1. **AI 领域热点** - 大模型、生成式 AI、AI 应用等最新动态
2. **科技行业新闻** - 科技公司动态、产品发布、政策法规等
3. **学术研究进展** - 重要的论文发表、技术突破等
4. **行业分析** - 专家观点、市场趋势、投资动向等

## 使用方法

运行 `/每日热点` skill 命令，系统将：
1. 搜索当天 AI 及科技领域的热点新闻
2. 整理成结构化的中文简报
3. 保存到本目录（文件路径：News/年/月/日热点.md）
4. 自动更新索引文件

## 索引更新

索引文件 `INDEX.md` 会在每次生成简报时自动更新，包含：
- 所有简报的日期列表
- 每日简报的关键词标签
- 快速导航链接
