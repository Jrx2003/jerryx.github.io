# 高频交易 - 分类说明

> 更新日期：2026-03-15
> 分类目的：研究市场微观结构、订单簿建模、算法执行

---

## 重点研究方向

### 1. 市场微观结构
- 订单簿动态建模
- 市场冲击与流动性
- 信息不对称与逆向选择
- 价格发现机制

### 2. 订单簿分析
- Level-2数据建模
- 订单流预测
- 买卖压力指标
- 限价订单簿可视化

### 3. 算法执行
- VWAP/TWAP策略
- 最优执行算法
- 市场冲击最小化
- 智能订单路由

### 4. 高频策略
- 做市策略
- 统计套利（高频）
- 事件驱动策略
- 延迟套利

### 5. 机器学习应用
- 限价订单簿预测
- 市场冲击建模
- 交易信号生成
- 执行优化
